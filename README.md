## Overview
A web application that recommends songs based on user mood using Spotify Web API.

## Features
- Mood selection
- Real-time Spotify recommendations
- Album thumbnails
- Direct Spotify links

## Tech Stack
- Python
- Streamlit
- Spotify Web API (Spotipy)

## How to Run
1. Clone repo
2. Install requirements
3. Add .env file with Spotify credentials
4. Run: streamlit run app.py

- Spotify login (OAuth)
